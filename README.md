# 스터디 목적으로 생성
- 필요파일   
apikey.properties 파일 생성(root 위치)   

- 필요내용   
네이버 Maps Api 사용을 위한 키값   
전월세 실거래가 조회를 위한 키값   
예)  
KEY_NAVER_CLIENT_ID="키값"  
KEY_NAVER_CLIENT_SECRET="키값"  
KEY_APT="키값"  

- 제약사항  
공부목적으로 수원시 영통구 1개 지역만 호출하도록 되어 있음
